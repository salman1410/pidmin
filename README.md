### PIDMIN
admin finder powered by python

### REQUIREMENTS
* `python`(Version 2 or 3)
* `pip`(Python Package Manager)
* `requests`(Python Module). If Missing `pip install requests`

### SETUP & USAGE
* `git clone https://github.com/salman1410/pidmin/`
* `cd pidmin`
* `python pidmin.py`
